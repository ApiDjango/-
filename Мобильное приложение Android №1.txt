Цель: создать мобильное приложение для Android-устройств, позволяющее пользователю удобно и быстро осуществлять заданные функции.

Описание: Требуется разработать мобильное приложение для Android-устройств, позволяющее пользователю осуществлять следующие функции:

    Регистрация и авторизация пользователей
    Отображение списка товаров, доступных для покупки
    Поиск товаров по ключевым словам
    Отображение детальной информации о товаре, включая фотографии, описание, цену и наличие на складе
    Добавление товара в корзину
    Просмотр корзины и изменение количества товаров в ней
    Оформление заказа и оплата товаров с помощью платежной системы

Требования к приложению:

    Приложение должно иметь интуитивно понятный интерфейс и быть удобным в использовании.
    Дизайн должен соответствовать современным тенденциям и быть адаптированным к различным размерам экранов Android-устройств.
    Приложение должно работать быстро и стабильно на различных версиях операционной системы Android, начиная с версии 5.0.
    Приложение должно поддерживать работу с базой данных для хранения информации о товарах, заказах и пользователях.
    Приложение должно поддерживать безопасную передачу данных между клиентом и сервером.
    Приложение должно иметь возможность работы в оффлайн-режиме с локальным кэшем товаров и корзины.

Технические требования:

    Язык программирования - Java или Kotlin.
    Для работы с базой данных необходимо использовать ORM-фреймворк (например, Room).
    Для обмена данными с сервером необходимо использовать REST API и JSON-формат.
    Для реализации платежной системы можно использовать готовое решение (например, Stripe).
    Код приложения должен быть написан с использованием принципов чистой архитектуры и MVP-паттерна.
    Для работы с изображениями необходимо использовать библиотеку Picasso или Glide.

Ожидаемый результат:

    Исходный код приложения, разработанный в соответствии с требованиями и техническими спецификациями.
    APK-файл для установки приложения на устройства с операционной системой Android.
