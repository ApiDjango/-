Задача:

Написать Python-скрипт для обработки данных из CSV-файла и сохранения результата в базе данных.
Требования:

    Скрипт должен принимать на вход путь к CSV-файлу и название таблицы в базе данных, в которую будут сохранены данные.

    Скрипт должен считывать данные из CSV-файла и проводить их обработку:
        удаление дубликатов
        удаление пустых строк
        конвертация числовых значений в числовой формат
        форматирование дат в удобный для базы данных формат

    Обработанные данные должны быть сохранены в базе данных в указанную таблицу. Если таблицы не существует, то скрипт должен ее создать.

    Скрипт должен обрабатывать исключения при работе с файлами и базой данных.

    Скрипт должен быть написан на языке Python версии не ниже 3.6.

    Скрипт должен быть написан с использованием библиотеки для работы с CSV-файлами и библиотеки для работы с базой данных.

    Скрипт должен иметь комментарии в коде для лучшей читаемости.

Результат:

В результате выполнения скрипта должны быть обработаны данные из CSV-файла и сохранены в базу данных в указанную таблицу в удобном формате. 
	В случае возникновения ошибок должны быть выведены соответствующие сообщения.
