Задача:

Написать Python-скрипт для создания API-сервиса, который позволяет пользователям получать информацию о фильмах и сохранять их в избранное.
Требования:

    Скрипт должен использовать фреймворк Flask для создания RESTful API.

    Сервис должен предоставлять следующие эндпоинты:
        /movies - получение списка всех фильмов
        /movies/<id> - получение информации о фильме по ID
        /favorites - получение списка избранных фильмов
        /favorites/<id> - добавление/удаление фильма в/из избранного

    Скрипт должен использовать API-сервис для получения информации о фильмах. Например, можно использовать API сервиса The Movie Database (TMDb).

    Скрипт должен использовать базу данных для хранения информации об избранных фильмах.

    Скрипт должен обрабатывать исключения при работе с API-сервисом и базой данных.

    Скрипт должен быть написан на языке Python версии не ниже 3.6.

    Скрипт должен быть написан с использованием библиотек Flask, requests, и библиотеки для работы с базой данных.

    Скрипт должен иметь комментарии в коде для лучшей читаемости.

Результат:

В результате выполнения скрипта должен быть создан API-сервис, который позволяет пользователям получать информацию о фильмах и сохранять их в избранное. 
	Сервис должен быть доступен по указанному вами порту и IP-адресу. В случае возникновения ошибок должны быть выведены соответствующие сообщения.
