Задача: Разработать REST API на Java для онлайн магазина.

Требования к API:

    API должно предоставлять возможность создания, чтения, обновления и удаления товаров.
    Каждый товар должен содержать следующие поля: идентификатор, название, описание, цена, количество на складе, категория товара.
    API должно предоставлять возможность создания, чтения, обновления и удаления категорий товаров.
    Каждая категория должна содержать следующие поля: идентификатор, название и описание.
    API должно быть защищено авторизацией и аутентификацией пользователей.
    API должно использовать формат передачи данных JSON.
    API должно поддерживать CRUD-операции (создание, чтение, обновление и удаление) товаров и категорий товаров.
    API должно предоставлять возможность фильтрации и сортировки товаров по различным полям, например, по цене, названию или категории.
    API должно обрабатывать ошибки и возвращать соответствующие коды ошибок и сообщения.
    API должно быть документировано.

Технические требования:

    Язык программирования: Java 8+.
    Фреймворк: Spring Boot.
    СУБД: PostgreSQL.
    Все зависимости должны быть управляемы с помощью Maven или Gradle.
    Для тестирования API необходимо использовать JUnit.
    Для документирования API использовать Swagger или OpenAPI.

Результаты проекта:

    Исходный код приложения на GitHub.
    Документация API в формате Swagger или OpenAPI.
    Примеры запросов и ответов на основные API-методы.
    Все зависимости должны быть управляемы с помощью Maven или Gradle.
    Возможность быстрого и простого развертывания API на локальном сервере с помощью Docker или других средств контейнеризации.
